# English-Dictionary
I have created a simple English Dictionary using Python Django and PostgreSQL.
This repo consists all the necessary files for making an english dictionary. I ensure that the code are simple and easily understandable. 

Programming Language: Python
Framework: Django
Database: PostgreSQL

Features
1. It has a simple UI.
2. Large number of unique words.
3. Provides meaning of the word searched instantly.
4. Also gives synonyms and antonyms.
5. Easy to add, edit or remove new words.

Things that i learned from this project:
1. Django integration with Python.
2. Models creation in Django and data insertion process.
3. Using external database storage mechanism PostgreSQL in VS code.
4. SQL queries and retrival of stored dataset.
